1) O que é semântica?

Semântica é o estudo do significado. Incide sobre a relação entre significantes, tais como palavras, frases, sinais e símbolos, e o que eles representam, a sua denotação. A semântica linguística estuda o significado usado por seres humanos para se expressar através da linguagem.

2) O que é semântica HTML (semântica no contexto da marcação HTML)?

A semantica no site HTML ajuda a deixar ele mais bem escrito e simples de ser lido e interpretado, ajudando até mesmo em sua busca no Google e facilitando o entendimento de leitores de acessibilidade. Com o HTML semântico, ficou muito mais fácil de interpretar páginas.

3) O que são elementos semânticos HTML? Dê exemplos de elementos semânticos e não semânticos.

Hoje, com o HTML5, temos a opção de dividir melhor as tags. E, utilizando o próprio HTML5 semântico, ficou ainda mais simples a compreensão para o computador.

Como exemplo temos as tags:
ARTICLE : A tag article, é um artigo dentro do conteúdo.
FOOTER : Temos também a tag footer que,é a tag utilizada para o rodapé da página ou até de rodapé para artigos.
FIGURE : A tag figure é utilizada para a inserção de figuras na página. Juntamente com essa tag trabalhamos com o alt, onde descrevemos o que há na imagem, facilitando então a acessibilidade da página, para os leitores de tela.

4) Cite alguns dos benefícios de se utilizar elementos semânticos (marcação semântica).

Ele tem maior acessibilidade. Você não é o único que acha que elementos semânticos são mais fáceis de entender. Os algoritmo de indexação em sites de busca e as tecnologias assistivas (como leitores de tela para usuários com deficiência visual) também entenderão melhor o contexto e conteúdo do seu website, entregando uma melhor experiência para os usuários. Geralmente, elementos semânticos também resultam em um código mais consistente.
